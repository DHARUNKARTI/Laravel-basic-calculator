# Laravel Calculator via Routing

A beginner Laravel project that demonstrates how routing works by building a simple calculator directly through URL routes and Blade views.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Dynamic route parameters
- Blade templating
- Layout inheritance
- Basic Laravel routing concepts

---

# Tech Stack

- PHP 8.x
- Laravel
- Blade Templating Engine

---

# Project Purpose

This project was built to understand and practice:

- Laravel routing
- Route parameters
- Controllers
- Blade views
- Layout inheritance
- Dynamic data rendering
- Basic backend request flow
